    Proxy configuration dialog does not work.

Go to Edit/Preferences... and bring up the Prefs UI that allows you to configure
 Mozilla behavior.

Select Advanced / Proxies to configure a proxy server.

Problems:

1.  Only one radio button should be checked at any time.  Currently, selecting
an unselected radio button does not cause other radio button(s) to become
unselected - this makes no sense of course, since there is only valid proxy
config for a particular installation.  It is possible to have all three radio
buttons turned on.

2.  For Manual proxy configuration:

No clickable button (in Netscape 4 it was named "View...") which allows you to
actually enter the details of your proxy configuration such as server and port.
