    IMAP:  Unsubscribed parent w/child folders is not recognized

1) In 4.x, I have a parent folder (top level) with child folders.  The parent
folder is not subscribed to.

2) In 5.0, the parent folder and the child folder are not displayed in the
folder pane.  In the IMAP log file, the line:

* LSUB (\HasChildren)

is missing for the parent folder although all the child folders are listed.

We should behave the same way as 4.x where the parent folder doesn't have to be
subscribed to if there are child folders.
