    * TITLE/SUMMARY
[PP] Color Mozillazine graphic (GIF) has white-on-black (1-bit) colorization.

* STEPS TO REPRODUCE
0) Using the 1.4.98 build of NGLayout, go to http://www.mozillazine.org or http:/
/www.mozillazine.org/image/title.gif

* RESULT
 - What happened

The MozillaZine image appears as a black rectangle, with the web site name of
"MozillaZine" appearing simply as dotted border in white. (Hard to explain, but
easy to see.)

Normally, the background should be a gradient orange-like color steadily
diminishing in intensity towards the bottom of the rectangle, with the word
"MozillaZine" in solid black text.

Occurs on 8-bit, 16-bit and 24-bit video. (as set prior to the launch of
viewer.app, assuming it won't handle dynamic changes while running.)

 - What was expected

Image to be displayed properly.

* REGRESSION

 - Occurs On
        viewer (1.4.98 build for Mac OS)

 - Doesn't Occur On
        viewer (1.4.98 build for Win32)
        Navigator 3.04 RTM for Mac OS


* CONFIGURATIONS TESTED

- PowerMac 8500/150 (233 Mhz 604e), 64 MB RAM, Mac OS 8.5.1
