    Component checklist is hooked into config.ini and IB is hooked into checklist.
Only selected components are included in the build.  That is, the checklist
contains an entry for every component described in the config.ini but only those
that remain checked when the "build installers" button is pressed will be copied
into the output.  (NB: core.xpi will get included even if it's not checked since
we customize it to contain things collected in the wizard.)
