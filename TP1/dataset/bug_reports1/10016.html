    so this is no longer needed to fix for m9. I updated the xul to make the tooltip
appear at the bottom right of the button (not under the mouse).

popupanchor attribute or sets it to "topLeft", it will still crash because the
tooltip will go away too quickly. Therefore, this is still a problem, but it's
not critical.
