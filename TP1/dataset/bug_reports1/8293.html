    data point:  its Alt+D for all cases on the XFE in 4.x (even cancel message.)

the check for author is done at cancel time:  either by the server (if it
supports that extension) or by our application, it loops through the identities
to see if one of them matches the author of the article.

I'd be all for making it Del for all platforms all the time.
(And, once we support alternate hot keys for alternate platforms, making it
Alt+D for UNIX, and something else, if necessary, for the Mac.)

Until we have that infrastructure, I'd just make it Del for all platforms, in
all cases, and move on.
