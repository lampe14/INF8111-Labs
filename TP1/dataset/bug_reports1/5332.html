    This is an enhancement request from 4.5 product line that the customer would
like to see in 5.0.

On Windows platform in 4.x, there is a File | Page Setup menu item that allows
the user to enable and disable printing of a) document title, b) document
location URL, c) page number, d) page total, and e) date printed.

On Unix platform in 4.x, there is no File | Page Setup menu item.  So, the user
does not have the option to disable these items during printing.

This bug report is for the ability to enable/disable these items during printing
in 5.0.

If this isn't clear, pls let me know and I'll try to explain further.
