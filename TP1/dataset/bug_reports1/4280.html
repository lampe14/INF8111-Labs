    Allowing/testing for multiple instances of CCK can get complicated.  What if a
person is using two different CCK instances on the same configuration?  Which
one would override the other?  Also, what if the end user runs the build process
at the same time on the same configuration?  Will there be a permission error?

If the build process took 10 or more minutes, I could see the need to trigger
multiple builds to save time.  However, I believe it only takes a few minutes so
I do not see the need to support multiple instances.

Also, the more important question has been already asked in this thread.  Is it
easier to enforce a single instance or support multiple instances?  Our customer
only needs one instance.
