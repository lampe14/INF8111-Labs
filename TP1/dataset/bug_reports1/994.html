    We don't yet support HTML 4.0's OPTGROUP element. Here's how it should work:

 <SELECT name="ComOS">
        <OPTGROUP label="Fruits">
          <OPTION>Apple
          <OPTION>Orange
          <OPTION>Pineapple
        </OPTGROUP>
        <OPTGROUP label="Meats">
          <OPTION>Beef
          <OPTION>Chicken
        </OPTGROUP>
        <OPTGROUP label="Deserts">
          <OPTION>Ice Cream
          <OPTION>Cake
        </OPTGROUP>
    </SELECT>

Should render as:

[ Fruits  ]
[ Meats   ]
[ Deserts ]

When you mouseover "Deserts" for example, you get:

[ Fruits  ]
[ Meats   ]
[ Deserts >  Ice Cream ]
          [   Cake     ]
