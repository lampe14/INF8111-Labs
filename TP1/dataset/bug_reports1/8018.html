function name change):

    Code nit, really don't mean to be annoying, just trying to promote "good style":

  if (foo == bar)
    return true;
  else
    return false;

where true is a synonym for 1 and false for 0, is a long way of saying

    return foo == bar;

Note also that else after return is a non-sequitur, and however comforting, is
not worth the over-indentation (and possible confusion).
